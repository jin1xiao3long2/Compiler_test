PUSH：
    PUSH 1024 (push立即数)
    PUSH 1 1 (push变量栈内元素：方向（0正），偏移量)
PDAT 1(push数据段元素：偏移量) 
COPY 0 0 (回填栈顶元素->变量栈：同上)
SPOP 0 0 4 (弹栈：方向（0正），方式（0次数弹栈/1弹栈至第几位），值)
FLAG name (注册标记(同一域内))
CJMP/NJMP name (跳转至标记)(C:>0则跳、N:无条件跳)
SADD (栈顶+次栈顶)
DADD 1 (栈顶+数据段[数据偏移])
IADD 1024 (栈顶+立即数)
RETP (返回函数)
CALL 2 fun_name (调用)
